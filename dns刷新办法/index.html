Linux 刷新 dns 缓存

  1. 刷新DNS－Systemd Resolved
1

systemd-resolved --flush-caches
  1. 刷新DNS－NSCD
1

systemctl restart nscd.service

windows 刷新 dns 缓存

1

ifconfig/flushdns

Mac 刷新 dns 缓存

1

sudo dscacheutil -flushcache;sudo killall -HUP mDNSResponder
