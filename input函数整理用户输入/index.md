# 【Golang】自定义 input 函数整理用户输入


<!--more-->

把 fmt.Scan 用户输入整理成函数，方便底层调用，重复代码

```go
package main

import (
	"fmt"
)

func input(prompt string) string {
	var text string
	fmt.Print(prompt)
	fmt.Scan(&text)
	return text
}

func main() {
	a := input("你输入的是啥：")
	b := input("还是 python 简单很多：")
	c := input("学会 go 好处多多多哦：")
	fmt.Printf("%v, %v, %v", a, b, c)
}
```
